# Brother Printer Setup

## Overview
I set up a network printer in a home environment.

## Problem
The printer showed IP address 0.0.0.0 and could not connect to the network.

## What I did
- Opened the network settings on the printer
- Used the setup assistant to connect to Wi-Fi
- Checked the router connection
- Added the printer manually via TCP/IP on Windows

## Result
The printer received an IP address (192.168.x.x) and printed successfully.

## What I learned
- How devices get an IP address via DHCP
- How to troubleshoot network connection issues

# Mobile Data Issue (Congstar / T-Mobile)

## Overview
I troubleshooted a mobile internet issue on a smartphone using the Congstar (T-Mobile) network.

## Problem
Mobile data was not working correctly and internet speed was extremely slow (approx. 10 Kbit/s).

## Cause
- No available credit on the SIM card
- Incorrect APN configuration

## What I did
- Checked mobile network settings
- Verified mobile data connection
- Reviewed and adjusted APN settings (Access Point Name)
- Recharged mobile credit

## Result
Mobile internet connection was restored and worked normally again.

## What I learned
- How mobile networks depend on APN configuration
- Importance of checking both technical settings and account status
- Systematic troubleshooting approach (problem → cause → solution)
